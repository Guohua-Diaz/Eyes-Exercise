# Eyes-Exercise

## Two eyes movement
```What should you expect on this game?  The two eyes would follow your mouse movement accordingly. You can move your mouse as desired.```

## How to run 
1. Clone the repo
2. Open Chrome 
3. On Chrome, click file -> open File
4. Select index.html
